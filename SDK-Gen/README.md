# Spotify Music Radar

A small web app for browsing popular Spotify artists and tracks by year. It defaults to the current year, lets you search another year, opens artist top tracks in a side panel, and links every track to Spotify.

## Setup

1. Create a Spotify app in the Spotify Developer Dashboard.
2. Copy `.env.example` to `.env`.
3. Fill in:

```bash
SPOTIFY_CLIENT_ID=your_client_id
SPOTIFY_CLIENT_SECRET=your_client_secret
```

4. Install and run:

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

The server uses the generated SDK in `spotify-sdk` and keeps Spotify credentials out of the browser.
