# FollowPlaylistRequest

**Properties**

| Name   | Type    | Required | Description                                                                                                                                                                                                                                                 |
| :----- | :------ | :------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| public | boolean | ❌       | Defaults to `true`. If `true` the playlist will be included in user's public playlists (added to profile), if `false` it will remain private. For more about public/private status, see [Working with Playlists](/documentation/web-api/concepts/playlists) |
