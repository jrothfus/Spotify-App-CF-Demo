# ExplicitContentSettingsObject

**Properties**

| Name          | Type    | Required | Description                                                                                          |
| :------------ | :------ | :------- | :--------------------------------------------------------------------------------------------------- |
| filterEnabled | boolean | ❌       | When `true`, indicates that explicit content should not be played.                                   |
| filterLocked  | boolean | ❌       | When `true`, indicates that the explicit content setting is locked and can't be changed by the user. |
