
Built the Spotify Music Radar web app and started it locally.

Open it here: http://localhost:5173

What’s included:

Current-year default view with year search.
Ranked Spotify artists and tracks.
Click an artist to load their top tracks.
Click any track to open it in Spotify.
Express API keeps Spotify credentials server-side and uses SDK-Gen/spotify-sdk.
Setup note: the app is running, but Spotify data needs credentials. Add a .env from .env.example:

SPOTIFY_CLIENT_ID=...
SPOTIFY_CLIENT_SECRET=...
Then restart with:

npm run dev
Verification: npm run build passes, the API health endpoint responds, and the Vite UI is serving successfully.
